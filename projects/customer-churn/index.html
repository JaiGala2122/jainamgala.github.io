Executive Summary

Business Problem: A European bank experiences 20% annual customer churn, resulting in significant revenue loss. This project develops a machine learning model to identify at-risk customers before they leave.

85% Model Accuracy (ROC-AUC)
10K Customers Analyzed
3 Key Risk Factors

The Challenge

Banks invest heavily in customer acquisition, but retaining existing customers is more cost-effective. The dataset revealed:

  • 20% churn rate - 1 in 5 customers leave annually
  • Class imbalance - Only 20% positive cases makes prediction difficult
  • Multiple factors - Age, geography, products, and balance all play a role

Key Findings

Finding #1: Customers aged 41-50 have a 45% churn rate - more than double the average.
Finding #2: Female customers churn at 25% compared to 16% for males - a significant gender gap requiring investigation.
Finding #3: Customers with 3+ products unexpectedly show higher churn, suggesting possible over-selling or dissatisfaction.

Technical Approach

1. Data Preprocessing

  • Cleaned dataset of 10,000 customers with 14 features
  • Encoded categorical variables (Geography, Gender)
  • Handled class imbalance using SMOTE oversampling
  • Scaled numerical features using StandardScaler

2. Feature Engineering

# Created new features: - BalanceToSalary: Ratio of account balance to estimated salary - TenureAgeRatio: Relationship between tenure and age - IsZeroBalance: Binary flag for zero balance accounts - AgeGroup: Categorical age brackets

3. Model Selection

Tested three algorithms:

  • Logistic Regression (Baseline): 79% ROC-AUC
  • Random Forest: 83% ROC-AUC
  • Gradient Boosting (Selected): 85% ROC-AUC
[Insert image: roc_curves.png]
ROC curves comparing all three models

4. Model Evaluation

Used ROC-AUC instead of accuracy because:

  • Handles class imbalance better than accuracy
  • Measures model's ability to distinguish between classes
  • More reliable for business decision-making

Results & Impact

Model Performance

  • 85% ROC-AUC score on test data
  • 82% precision in identifying churners
  • Validated with 5-fold cross-validation

Top Predictive Features

  1. Age - Strongest predictor of churn
  2. Number of Products - Non-linear relationship
  3. Geography - Germany shows higher churn
  4. Balance - Zero balance indicates risk
  5. Gender - Significant difference between groups

Business Recommendations

Immediate Actions:

  1. Target 41-50 age group with retention campaigns - they represent the highest risk segment
  2. Investigate gender disparity - understand why female customers churn at higher rates
  3. Review product bundling - customers with 3+ products may be over-served or dissatisfied
  4. Monitor zero-balance accounts - implement early warning system for inactive accounts

Long-term Strategy:

  • Deploy model to score all customers monthly
  • Create tiered intervention based on churn probability
  • A/B test retention offers on high-risk customers
  • Measure ROI of retention vs acquisition spend

Challenges & Learning

Key Challenges:

  • Class imbalance: Initial model predicted "no churn" for everyone (80% accurate but useless). Solved using SMOTE and adjusting class weights.
  • Feature scaling: Forgot to scale features initially, which hurt model performance. Learned importance of preprocessing.
  • Metric selection: Started with accuracy, realized ROC-AUC is better for imbalanced data.

Technical Skills Gained:
